# Frontend

The React/Vite renderer for Shiftwork. This folder owns visual presentation, gearbox interaction, task/session state, local persistence, accessibility, and component/unit tests.

Run only the browser renderer from the repository root:

```bash
npm run dev:frontend
```

The source `index.html` is a Vite entrypoint and should not be opened directly with `file://`; use the command above or launch the complete desktop app with `npm run dev`.
